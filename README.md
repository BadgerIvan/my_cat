# MY_CAT
my_cat - small project for School 21 by Sber.

## Build
```bash
git clone https://github.com/BadgerIvan/my_cat.git
cd my_cat/src
make
```

## Use
```bash
./my_cat [OPTIONS] [FILE]...
```
Supported flags:
```
-b / --number-nonblank
-E / -e
-n / --number
-s / --squeeze-blank
-T / -t
-v
```

## Run tests
```bash
./my_cat_test
```
